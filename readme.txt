Bienvenu dans le github du projet SAMS (nom provisoire)

V1.0 :

- Envoi du message d'alerte vers un seul numéro de téléphone.
- Modification des numéros directement dans le code du script.

V1.1 :

- Envoi du message d'alerte vers plusieurs numéros de téléphone.
- Modification des numéros dans un fichier à part, par bip.

V2.0 :

- Ajout d'une interfaçe visuelle.
- Configuration depuis l'interfaçe.
- Test disponible depuis l'interfaçe.

INSTALL :

- install.sh : installation basique, des options sont disponible sur le script (--help)
- install_dial.sh : installation graphique, les options seront automatiquement proposées
